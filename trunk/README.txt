---------------------------
 Installation instructions:
---------------------------

For the impatient, experienced, and brave, here's the short version:

1) Install felix version 3.0.1 or later
2) Copy the directories under docs/felix/ to your felix installation
3) Install the 3rd party tools by executing the script docs/scripts/3rd_party/menu3p as root
4) Run "mvn install -DdeployTo=<path to felix>/matterhorn"
5) Start felix and navigate to http://localhost:8080

Detailed instructions for a variety of platforms and topologies are available at http://opencast.jira.com/wiki/display/MH/Install+and+Configure+%28Trunk%29

Questions should go to the Matterhorn list (matterhorn@opencastproject.org) or the #opencast IRC channel at irc.freenode.net
