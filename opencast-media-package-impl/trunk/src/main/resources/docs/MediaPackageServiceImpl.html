Media Package Service

$Revision$

Get a media package by its ID

An HTTP GET to /{id} will return a media package:

<media-package handle="sample">
    <track-list>
      <track>
        <url>http://localhost:8080/tracks/sample/track1</url>
      </track>
      <track>
        <url>http://localhost:8080/tracks/sample/track2</url>
      </track>
    </track-list>
  </media-package>

Get all media packages

An HTTP GET to /all will return a media package list (note: this will need to be changed or removed for production):

<media-packages>
      <media-package handle="listsample1">
        <track-list>
          <track>
            <url>http://localhost:8080/tracks/listsample1/track1</url>
          </track>
          <track>
            <url>http://localhost:8080/tracks/listsample1/track2</url>
          </track>
        </track-list>
      </media-package>
