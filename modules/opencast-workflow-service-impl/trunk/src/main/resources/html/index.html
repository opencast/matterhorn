Workflow Service REST endpoint

Workflow Definitions

GET defs returns all workflow definitions as a Workflow Definition List
GET def/{id} returns a workflow definition
POST def/{id} registers a new workflow definition, which should be passed in the "wfdef" form parameter

Workflow Instances

GET instances/{id} returns all instances for a given workflow definition ID as a Workflow Instance List
GET instance/{id} returns a workflow instance
POST start/{id} starts a workflow instance, based on the specified workflow definition ID and media package passed in the "mediapackage" form parameter
