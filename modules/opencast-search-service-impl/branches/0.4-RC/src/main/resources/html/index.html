Search Service REST endpoint

GET episode returns episodes that match these query parameters:
id The episode ID seriesId All episodes in the series with this ID q Any episode that matches this free-text query. For example: ./episode?q=vegetation limit The maximum number of items to return per page offset The page number
GET series returns series that match these query parameters:
id The series ID. This takes the additional boolean "episodes" parameter. If true, the result set will include this series episodes. q Any series that matches this free-text query.
